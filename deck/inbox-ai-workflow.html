Today's Problem

Re-engagement drives conversions, but Meta's current inbox tools don't make it easy for businesses to do it at scale

Businesses can only reply one by one — most leads never get a follow-up, driving them to third-party tools

>70%

conversations from ads

63%

conversations have no follow-up within 24 hours

~10%

use third-party tools for re-engagement

2.4×

conversion when re-engage

Proposal

A native re-engagement tool built into Meta Business Inbox

Enable businesses to re-engage existing customers at scale

MVP

  • Audience: Target active advertisers
  • Use Case: Ad conversations follow-up
  • Surface: Business inbox desktop

Long Term

  • Audience: All messaging businesses
  • Use Case: Post-purchase re-engagement, broadcast
  • Surface: Cross-platform rollout

Current Experience

Audit finding: Our tool only automates first questions/responses, while businesses need follow-up conversations to drive outcomes

First message sent through automation or ad setup

Learnings

Third-party platforms offer simple tool to send broadcast or bulk messages. Meta tested with broadcast tool before and failed.

Baseline

Used Figma to build the first version as a baseline for AI to continue exploring.
